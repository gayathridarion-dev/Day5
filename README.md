# 🎵 Chatbot Song Recommender

**Description:**
This project is an AI-powered Song Recommender system built with Python, Flask, Selenium, and NLP models. Users interact with a chatbot interface to get song recommendations based on their mood. The system fetches live song data from online platforms and analyzes user input using natural language processing.

**Key Features:**

* **Chatbot Interface** – Conversational UI where users describe their mood or song preference.
* **AI/NLP Mood Detection** – Detects user mood from text and suggests relevant songs.
* **Selenium Integration** – Automates search on music platforms to fetch real-time song recommendations.
* **Flask Backend** – Handles API endpoints and serves the frontend interface.
* **Branch-based Git Workflow** – Developed features in separate branches (`chatbot-feature`, `flask-feature`, `selenium-feature`) and merged into `main`.

**Tech Stack:**

* Backend: Python, Flask
* Automation: Selenium
* AI/NLP: Hugging Face Transformers / Custom Mood Detection Model
* Frontend: HTML, CSS, JavaScript
* Version Control: Git (feature branches and merges)

**How it Works:**

1. User sends a message to the chatbot describing their mood.
2. The AI/NLP model detects the mood from the text.
3. Selenium scrapes song recommendations from online music platforms.
4. Flask serves the frontend and displays the recommended songs to the user.

**Project Highlights:**

* Real-time song recommendations based on mood.
* Fully modular design with separated chatbot logic, AI model, and Selenium automation.
* Professional Git workflow with feature branches and merges.
* Ready to extend for personalization, analytics, or deployment.

**Future Enhancements:**

* User authentication for personalized playlists.
* Analytics to track popular moods and songs.
* Deployment to cloud platforms for global access.
