# 🎵 Chatbot Song Recommender System

This project is a full-stack **Song Recommender System** built using **Flask, Selenium, and AI/NLP**. It allows users to interact with a chatbot, detect their mood, and get song recommendations automatically.

---

## **Features**

### 1. Chatbot Feature
- Detects user queries and mood using an NLP model.
- Suggests songs based on user input and mood.
- Provides conversational experience.

### 2. Flask Feature
- Web-based interface for users to interact with the chatbot.
- Tracks analytics such as page views and clicks.
- Simple and responsive UI using HTML, CSS, JS.

### 3. Selenium Feature
- Automates song searches and gathers data from online sources.
- Integrates with the Flask backend to provide recommendations.
- Helps simulate real-world usage and testing.
